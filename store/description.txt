TabDraft - New Tab Markdown Editor

Turn your new tab into a distraction-free writing workspace. TabDraft replaces Chrome's default new tab page with a Markdown editor that auto-focuses on open.

Features:
- Markdown editor with syntax-aware styling (headings, bold, italic, links, code)
- Tab/Shift-Tab to indent/outdent list items
- Optional TODO list with completion tracking
- 4 built-in themes: System (auto), Light, Dark, Monokai
- Fully customizable theme via CSS variables
- Adjustable font size, heading style, and text alignment
- Auto-focus on editor when opening a new tab
- Bilingual UI: English and Japanese (auto-detected)
- Data persists locally via Chrome storage

Keyboard Shortcuts (customizable):
- Alt+1: Focus TODO list
- Alt+2: Focus Markdown editor

Your data stays local in your browser. No accounts, no cloud sync, no tracking.
